## Security Policy

### Supported Versions

Only the latest stable release of Royal Blueprint receives security updates. Always keep your installation up to date.

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| older   | ❌ |

### Reporting a Vulnerability

To report a security vulnerability, please contact **Shaurya Vashishtha (Royal Devlopments)** directly via GitHub Issues or reach out through the repository discussions.

Please refrain from publicly disclosing vulnerabilities until they have been addressed.
